## Literature management tool v1.1.1

这是一个针对 `v1.1.0` 的热修复版本，主要解决设置对话框中文文案显示异常的问题，确保正式发布包中的“跨设备同步”设置页可正常使用和阅读。

### Release 页面

- Release 页面：[v1.1.1](https://github.com/Dongyurocket/literature-management-tool/releases/tag/v1.1.1)
- 安装包：`Literature-management-tool-v1.1.1-Setup.exe`

### 本版重点

- 修复设置对话框标题、字段标签、元数据源名称和说明文本显示为 `??` 的问题
- 恢复“跨设备同步”区域的完整中文说明，避免正式包内出现不可读界面
- 增加 Qt 回归测试，校验设置页关键中文文案保持可读

### 验证情况

- `python -m pytest -q`
- `python -m compileall main.py literature_manager tests`

### 说明

- `v1.1.1` 延续 `v1.1.0` 的跨设备同步工作区能力
- 如果你已经下载了 `v1.1.0` 安装包，建议升级到 `v1.1.1`
