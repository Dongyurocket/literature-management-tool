## Literature management tool v0.2.2

本次发布重点完善了公开仓库展示、Windows 安装体验，以及中文 Release 文档。

### 版本亮点

- 新增 Windows 安装版 `Setup.exe`，支持安装向导、开始菜单快捷方式、桌面快捷方式和卸载入口
- 仓库补充 `MIT License`，更适合公开发布和二次使用
- README 重写为更完整的中文说明，补充了安装、工作流、打包和数据存储说明
- Release 补充中文说明与界面截图，便于首次访问仓库时快速了解产品
- GitHub Actions 发布流程已改为自动上传 `Setup.exe`

### 适合谁使用

- 需要本地优先管理 PDF、翻译稿、补充材料、笔记的研究者
- 需要维护 GB/T 7714-2015 相关字段和 BibTeX 导出的用户
- 希望把 `docx / md / txt` 笔记和文献条目统一管理的团队或个人

### 安装说明

1. 下载本页附件 `Literature-management-tool-v0.2.2-Setup.exe`
2. 双击运行安装程序
3. 按向导选择安装位置和快捷方式选项
4. 安装完成后即可从开始菜单启动 `Literature management tool`

> 卸载程序不会自动删除你的本地文献数据库和文献库文件。

### 主要功能回顾

- 自定义文献库存储目录，并支持复制 / 移动 / 仅关联三种导入方式
- 文献条目支持原文、翻译、补充材料、多个笔记文件同时关联
- 支持 `docx` 文件笔记与自定义 PDF 阅读器
- 支持批量导出 BibTeX、CSL JSON 与国标参考文献文本
- 支持 PDF 自动命名、全文检索、查重合并、备份恢复、路径修复

### 已知说明

- 当前版本仍未包含 OCR
- PDF 元数据抽取与全文提取属于尽力而为能力
- 若已有旧版绿色包，可继续使用原数据目录，安装版不会破坏已有数据库
