## Literature management tool v0.2.3

这次发布对应 Phase 4 的核心交付：旧版文献管理能力已经完整迁到 Qt，新界面不再依赖 Tkinter，并补齐了后台任务、维护工具和安装包验证。

### 下载与校验

- Release 页面：[v0.2.3](https://github.com/Dongyurocket/literature-management-tool/releases/tag/v0.2.3)
- 安装包：[Literature-management-tool-v0.2.3-Setup.exe](https://github.com/Dongyurocket/literature-management-tool/releases/download/v0.2.3/Literature-management-tool-v0.2.3-Setup.exe)
- SHA256：`BB31918C858389339C9BAEEF32305E189C8E8A4468DF8C8D46F3784F808F8A9B`

### 版本亮点

- Qt 成为唯一 GUI 入口，旧 `tkinter` 界面和相关遗留文件已移除
- 导入、查重、备份恢复、PDF 批量重命名等旧功能已完整迁到 Qt 工具面板
- 拖拽导入、DOI / ISBN 元数据抓取改为后台任务，长操作不再阻塞界面
- 新增线程池与非阻塞 toast 提示，任务反馈更清晰
- 保留并完善 `docx` 笔记、自定义 PDF 阅读器、全文检索、路径修复等 V2 能力
- 已重新生成并验证 Windows 安装包 `Setup.exe`

### 适合谁使用

- 希望在本地长期维护 PDF、翻译稿、补充材料和笔记的研究者
- 需要管理 GB/T 7714-2015 字段并导出 BibTeX / CSL JSON 的用户
- 希望把 `docx / md / txt` 笔记与文献条目统一管理的个人或小团队

### 安装说明

1. 下载本页附件 `Literature-management-tool-v0.2.3-Setup.exe`
2. 双击运行安装程序
3. 按安装向导选择安装位置与快捷方式
4. 安装完成后，从开始菜单启动 `Literature management tool`

> 卸载程序不会自动删除你的本地文献数据库和文献库文件。

### 本次重点能力回顾

- 自定义文献库存储目录，并支持复制 / 移动 / 仅关联三种导入方式
- 文献条目支持原文、翻译、补充材料、多个笔记文件同时关联
- 支持 `docx` 文件笔记与自定义 PDF 阅读器
- 支持 PDF 批量重命名预览与执行
- 支持全文检索、查重合并、备份恢复、路径修复和索引重建
- 支持 BibTeX、CSL JSON 与国标 GB/T 7714 文本导出

### 已知说明

- 当前版本仍未包含 OCR
- PDF 元数据抽取与全文提取属于尽力而为能力
- 查重合并仍建议人工确认后执行
- 若已有旧版绿色包，可继续使用原数据目录，安装版不会破坏已有数据库
