## Literature management tool v0.3.3

这一版是稳定性热修复，重点修复“检查更新偶发 HTTP 403 导致失败”的问题，并补充更新链路回退提示。

### 主要修复

- 修复：检查更新在 GitHub API 返回 `HTTP 403/429` 或网络错误时，自动回退到 Release 网页解析，不再直接失败。
- 增强：支持读取 `GITHUB_TOKEN` / `GH_TOKEN`，可在受限环境下提升更新检查成功率。
- 优化：更新结果中增加来源标记（API / Web fallback）。
- 优化：更新说明弹窗新增“回退提示”，用户可直观看到当前使用的检查路径。
- 测试：补充更新检查回退与错误兜底用例，防止回归。
