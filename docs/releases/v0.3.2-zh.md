## Literature management tool v0.3.2

这一版是稳定性修复发布，重点解决“命令已完成但右下角仍显示正在运行”的交互问题，并补上文献列表手动刷新能力。

### 下载与校验

- Release 页面：[v0.3.2](https://github.com/Dongyurocket/literature-management-tool/releases/tag/v0.3.2)
- 安装包：`Literature-management-tool-v0.3.2-Setup.exe`
- SHA256：`8D7E11BD6AB390B0A2FA586D0501AF71B4E9B03869FA5AA634ED753FC52DB466`

### 本版重点改进

- 新增文献列表刷新功能（顶部 `刷新列表` 按钮 + `F5` 快捷键）
- 刷新前会先保存当前编辑中的元数据，避免刷新后编辑内容丢失
- 修复多处“任务已完成但状态栏仍显示进行中”的问题
- 增加 busy 状态守护机制：若无活动线程且状态残留，会自动恢复到 `就绪`
- 元数据抓取改为每次仅使用一个已选来源，避免一次触发多源混合回退
- 元数据合并策略增强：支持替换占位标题、作者去重合并、关键词去重合并
- 引用键（cite_key）改为可手动编辑并自动保存

### 测试与验证

- 单元测试：`python -m unittest discover -s tests -v`（43 项通过）
- 打包命令：`powershell -ExecutionPolicy Bypass -File .\scripts\build_windows.ps1 -Version 0.3.2`

### 升级说明

1. 下载并运行 `Literature-management-tool-v0.3.2-Setup.exe`
2. 按安装向导完成升级
3. 卸载程序默认不会删除你的文献数据库与文献库目录
