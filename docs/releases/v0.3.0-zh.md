## Literature management tool v0.3.0

这一版是面向中文研究工作流的一次增强发布：Qt 桌面端已全面中文化，并补上自动更新、Umi-OCR 扫描版 PDF 识别、导出模板、统计报表、多文库切换与更完整的重复对比体验。

### 下载与校验

- Release 页面：[v0.3.0](https://github.com/Dongyurocket/literature-management-tool/releases/tag/v0.3.0)
- 安装包：`Literature-management-tool-v0.3.0-Setup.exe`
- SHA256：`41E7CD0C4ECA568CBB0CC6E42F9F66108A5B9144351848DB3DBF1722A6D33618`

### 版本亮点

- 主界面、设置页、导入中心、统计页、工具对话框已统一切换为中文界面
- 支持软件内检查 GitHub Release 更新，并下载新版 `Setup.exe`
- 支持在设置中一键下载安装 `Umi-OCR`，自动读取 Umi HTTP 服务端口并执行扫描版 PDF OCR
- 保留自定义 OCR 命令模板，可按 `{umi_ocr}`、`{input}`、`{output}` 接入自定义脚本
- 元数据抓取回退链路扩展为 `Crossref / DataCite / OpenAlex / OpenLibrary / Google Books`
- 新增模板导出与统计报表导出，支持 `Markdown / CSV / HTML / JSON`
- 支持多文库切换与归档库管理，每个文库拥有独立数据库与设置文件
- 重复检测界面升级为字段级冲突对比与合并预览

### 安装说明

1. 下载本页附件 `Literature-management-tool-v0.3.0-Setup.exe`
2. 双击运行安装程序
3. 按中文安装向导选择安装位置与快捷方式
4. 安装完成后，从开始菜单启动 `Literature management tool`

> 卸载程序不会自动删除你的本地文献数据库和文献库文件。

### 本次重点能力回顾

- 自定义文献库存储目录，支持复制 / 移动 / 仅关联三种导入方式
- 文献条目支持原文、翻译、补充材料与多个笔记文件同时关联
- 支持 `docx / md / txt` 外部笔记与内置文本笔记混合使用
- 支持 PDF 批量重命名预览与执行
- 支持全文检索、查重合并、备份恢复、路径修复和索引重建
- 支持 BibTeX、CSL JSON、GB/T 7714 文本导出
