# 文献管理软件

一个离线优先的本地文献管理桌面应用，使用 `tkinter + sqlite3` 编写，适合在 Windows 上管理 PDF、翻译稿、笔记与 Bib 导出。

## 已实现的 V2 第一批功能

- 自定义文献库目录，可选择复制、移动或仅关联外部文件
- 文献条目支持 GB/T 7714-2015 常见字段，并扩展主题、关键词、一句话简介、摘要、备注、阅读状态、评分、标签
- 一条文献可关联多个附件：原文、翻译、笔记文件、补充材料
- 一个结构化笔记可绑定多个附件
- 支持将 `docx`、`md`、`txt` 作为外部笔记文件关联到文献，并继续绑定多个附件
- 支持为 PDF 配置自定义阅读器软件，打开 PDF 时优先使用该程序
- 支持多选文献导出 `.bib`
- 支持按 `作者_年份_标题_原文/翻译` 对 PDF 批量重命名
- 支持题名/作者/关键词/主题检索和按主题、年份、类型筛选

## 运行方式

```bash
python main.py
```

首次运行后，程序会在以下目录保存配置和数据库：

- 默认：`%APPDATA%\LiteratureManager`
- 可选：设置环境变量 `LITERATURE_MANAGER_HOME` 指向自定义目录

## 使用建议

1. 先在“设置”里指定文献库目录。
2. 新建文献，补充作者、年份、来源、主题、关键词等信息。
3. 选中文献后通过“添加附件”导入 PDF、翻译稿或补充材料。
4. 在右侧“笔记”页签中可创建文本笔记，或关联现有 `docx/md/txt` 笔记文件，并绑定相关附件。
5. 批量选择文献后使用“导出 Bib”或“PDF 重命名”。
